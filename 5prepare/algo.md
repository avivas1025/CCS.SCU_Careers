## ✏️笔试面试主要考察内容与形式

### 1. 笔试面试基本流程

算法岗的面试流程因公司而异，以下是常见的流程：

| 阶段 | 说明 | 注意事项 |
| :--- | :--- | :--- |
| **笔试** | 在线编程题 + 选择题，通常限时1-2小时 | 主要考察算法与数据结构、机器学习基础 |
| **AI面/机试** | 部分公司会有AI面试或在线编程测试 | 注意代码规范和注释 |
| **技术一面** | 基础知识 + 手撕算法题 | 偏重基础，如数据结构、ML/DL基础概念 |
| **技术二面** | 项目深挖 + 进阶算法题 | 会深入问项目细节和模型原理 |
| **技术三面/交叉面** | 综合能力考察 | 可能涉及系统设计、论文讨论等 |
| **HR面** | 聊薪资、职业规划、团队匹配度 | 诚实回答，展现诚意 |

> **小贴士**：大厂（如字节、腾讯、阿里等）通常有3-4轮技术面 + 1轮HR面；中小厂可能2轮技术面 + 1轮HR面。具体流程以公司通知为准。

### 2. 笔试主要考察内容与形式

**考察形式：**
- **在线编程题**：通常在牛客网、LeetCode 等平台上进行，要求在限定时间内完成 2-5 道算法题
- **选择题/填空题**：涵盖机器学习、深度学习、数学基础、编程语言基础等
- **编程语言**：一般支持 C++、Python、Java，推荐使用自己最熟练的语言

**主要考察内容：**
- **算法与数据结构**（占比最大）：数组、链表、树、图、动态规划、贪心、回溯等
- **机器学习基础**：常见算法原理（线性回归、逻辑回归、SVM、决策树、随机森林、XGBoost等）
- **深度学习基础**：CNN、RNN、Transformer、损失函数、优化器、正则化等
- **数学基础**：线性代数、概率论与数理统计、微积分（偏导数、梯度等）
- **编程基础**：Python/C++ 常用语法、标准库的使用

---

### 3. 面试主要考察内容与形式

**面试一般流程：** 自我介绍 → 项目深挖 → 基础知识问答 → 手撕代码 → 反问环节

**主要考察内容：**

| 考察方向 | 具体内容 |
| :--- | :--- |
| **手撕算法** | 现场在共享屏幕/白板上写代码，常见题型包括数组、链表、二叉树、动态规划、DFS/BFS等 |
| **项目深挖** | 面试官会针对简历上的项目提问，如"为什么选择这个模型？"、"遇到了什么困难？怎么解决的？" |
| **ML/DL基础** | "解释一下过拟合和欠拟合的区别及应对方法"、"Transformer的注意力机制原理"等 |
| **数学推导** | 可能要求推导损失函数的梯度、解释贝叶斯公式等 |
| **论文阅读** | 部分公司会问你是否读过某篇论文，或给你一篇论文摘要让你分析 |

### 4. 如何准备笔试&面试

#### 笔试准备

- **刷算法题（最重要）**：
  - 先刷 [LeetCode Hot 100](https://leetcode.cn/studyplan/top-100-liked/) 和 [面试经典 150 题](https://leetcode.cn/studyplan/top-interview-150/)
  - 按类别刷题：数组 → 链表 → 栈/队列 → 树 → 动态规划 → 图
  - 每道题先想暴力解，再想优化解；想不出来就看书解，理解后自己重写一遍
  - **重要题目隔一段时间重做**，保持手感
- **复习ML/DL基础**：参考 [5prepare/algo.md](./algo.md) 中的考察内容，系统复习
- **模拟笔试**：在牛客网上找往年的笔试题进行限时练习

#### 面试准备

- **准备自我介绍**：控制在2-3分钟，突出你的技术背景、项目经历和求职意向
- **熟悉自己的项目**：能流畅回答"为什么选这个模型"、"遇到了什么问题"、"结果如何"等
- **手撕代码练习**：每天至少练习1-2道题，保持编码速度和正确率
- **基础知识复习**：重点复习ML/DL的核心概念和常见面试题
- **模拟面试**：找同学互相模拟面试，练习表达和临场反应
- **了解目标公司**：提前了解公司的业务方向、技术栈，准备一些有针对性的问题
