## 📌技术学习路线-算法岗

> 本路线面向本科生同学，从基础到进阶，循序渐进。算法岗对数学和编程基础有一定要求，但不必畏惧，按部就班地学习即可。

#### 1. 整体路线概览

```
编程基础 → 数学基础 → 机器学习 → 深度学习 → 方向专精 → 项目实战
```

#### 2. 各阶段详细路线

##### 阶段一：编程基础（建议用时：1-2个月）

| 主题 | 详细描述 |
| :--- | :--- |
| **Python** | 算法岗最常用的编程语言。重点掌握：基础语法、NumPy（矩阵运算）、Pandas（数据处理）、Matplotlib（可视化） |
| **C++（可选）** | 部分公司要求用C++，尤其是推荐系统、自动驾驶等方向。掌握基础语法和STL |
| **数据结构与算法** | 这是算法岗面试的**核心**。必须掌握：数组、链表、栈、队列、哈希表、树（二叉树、BST、堆）、图、排序算法、二分查找、动态规划、DFS/BFS、贪心 |
| **Git** | 版本管理工具，学会基本的 clone、commit、push、pull、branch 操作 |

> **刷题建议**：从 [LeetCode](https://leetcode.cn/) 开始，先刷 Hot 100，再按专题刷题。每天坚持1-2题，重在理解思路而非死记代码。

##### 阶段二：数学基础（建议用时：1-2个月）

| 主题 | 详细描述 |
| :--- | :--- |
| **线性代数** | 向量、矩阵运算、特征值与特征向量、SVD分解、PCA原理。推荐：3Blue1Brown《线性代数的本质》 |
| **概率论与数理统计** | 概率分布（正态、均匀、伯努利等）、贝叶斯定理、最大似然估计、假设检验 |
| **微积分** | 偏导数、链式法则（反向传播的基础）、梯度、泰勒展开 |
| **最优化** | 梯度下降、牛顿法、拉格朗日乘数法（了解即可） |

> **学习建议**：不需要学得非常深，重点掌握在机器学习/深度学习中用到的数学概念。遇到不懂的数学公式时再回头查，效率更高。

##### 阶段三：机器学习（建议用时：2-3个月）

| 主题 | 详细描述 |
| :--- | :--- |
| **监督学习** | 线性回归、逻辑回归、SVM、决策树、随机森林、XGBoost/LightGBM、KNN |
| **无监督学习** | K-Means聚类、DBSCAN、PCA降维 |
| **模型评估** | 交叉验证、混淆矩阵、精确率/召回率/F1、ROC/AUC |
| **特征工程** | 特征选择、特征提取、缺失值处理、数据标准化/归一化 |
| **集成学习** | Bagging、Boosting、Stacking 的原理与区别 |

> **推荐课程**：
> - 吴恩达《Machine Learning》（Coursera / B站有中文字幕）
> - 李航《统计学习方法》（经典教材，建议重点阅读）
> - 《Scikit-Learn实战》

##### 阶段四：深度学习（建议用时：2-3个月）

| 主题 | 详细描述 |
| :--- | :--- |
| **基础概念** | 神经元、激活函数（ReLU、Sigmoid、Softmax）、前向传播、反向传播 |
| **框架使用** | **PyTorch**（推荐）或 TensorFlow，学会构建、训练、评估模型 |
| **CNN** | 卷积层、池化层、经典网络（ResNet、VGG、Inception）、图像分类、目标检测（YOLO系列） |
| **RNN/LSTM/GRU** | 序列建模、文本分类、时间序列预测 |
| **Transformer** | 自注意力机制、多头注意力、位置编码、BERT、GPT系列 |
| **训练技巧** | 学习率调度、Batch Normalization、Dropout、数据增强、迁移学习 |

> **推荐课程**：
> - 吴恩达《Deep Learning Specialization》（Coursera）
> - 李沐《动手学深度学习》（d2l.ai，强烈推荐，有中文版）
> - 斯坦福 CS231n（计算机视觉）、CS224n（自然语言处理）

##### 阶段五：方向专精（建议用时：2-4个月）

根据个人兴趣和就业方向，选择一个方向深入：

| 方向 | 核心内容 |
| :--- | :--- |
| **计算机视觉（CV）** | 图像分类、目标检测（YOLO、Faster R-CNN）、图像分割（U-Net、Mask R-CNN）、OCR、GAN |
| **自然语言处理（NLP）** | 文本分类、命名实体识别、机器翻译、文本生成、大语言模型（LLM）微调与部署 |
| **推荐系统** | 协同过滤、矩阵分解、DeepFM、Wide&Deep、序列推荐 |
| **数据挖掘/数据分析** | 用户画像、AB测试、因果推断、数据可视化 |
| **大模型/AIGC** | Prompt Engineering、RAG、LoRA微调、Agent开发、多模态模型 |

> **选择建议**：CV和NLP是算法岗需求最大的两个方向；推荐系统在互联网大厂需求稳定；大模型/AIGC是当前最热门的方向，但竞争也最激烈。

#### 3. 学习时间线建议

| 时间 | 建议内容 |
| :--- | :--- |
| 大一 | 编程基础（Python/C++）+ 数据结构与算法 + 数学基础 |
| 大二 | 机器学习 + 深度学习基础 + 开始刷LeetCode |
| 大三上 | 方向专精 + 做1-2个完整项目 + 参加竞赛 |
| 大三下 | 寻找实习 + 继续刷题 + 复习八股 |
| 大四上 | 秋招 + 完善简历和项目 |
