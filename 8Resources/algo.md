### 常见问题

#### 1. 常见的技术面/HR面的问题有哪些？有没有回答技巧？

**技术面常见问题：**

| 类别 | 常见问题 | 回答技巧 |
| :--- | :--- | :--- |
| **项目相关** | "介绍一下你最有成就感的项目" | 用 STAR 法则，重点讲你的贡献和成果 |
| **项目相关** | "为什么选择这个模型？有没有尝试过其他方案？" | 展示你的思考过程，说明对比实验的结果 |
| **项目相关** | "项目中遇到了什么困难？怎么解决的？" | 真实描述，体现你的问题分析和解决能力 |
| **ML基础** | "过拟合和欠拟合的区别？如何解决？" | 从原因和解决方案两个角度回答 |
| **ML基础** | "L1和L2正则化的区别？" | 从几何意义和实际效果两个角度解释 |
| **DL基础** | "解释一下Transformer的注意力机制" | 从Self-Attention的计算过程讲起 |
| **DL基础** | "Batch Normalization的原理和作用？" | 讲清楚计算过程和为什么能加速训练 |
| **手撕代码** | "实现快速排序/二叉树遍历/动态规划" | 先说思路，再写代码，注意边界条件 |

**HR面常见问题：**

| 问题 | 回答技巧 |
| :--- | :--- |
| "你为什么选择我们公司？" | 提前了解公司业务，结合自己的兴趣回答 |
| "你的职业规划是什么？" | 短期（1-2年）+ 长期（3-5年），体现稳定性 |
| "你有什么优缺点？" | 优点举具体例子，缺点说真实的但正在改进的 |
| "你期望的薪资是多少？" | 提前了解市场行情，给出合理范围 |
| "你还有什么想问我的吗？" | 准备2-3个问题，如团队技术栈、新人培养机制等 |

> **通用技巧**：回答问题时结构化表达（第一...第二...第三...），不会的问题坦诚说"这块我了解不深，但我的理解是..."，展示你的思考过程。
